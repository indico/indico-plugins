# Code Preview Plugin

This plugin adds syntax highlighting (using the Pygments syntax highlighter)
to event materials that look like source code files.

The following languages are supported:

- C / C++
- C#
- CSS
- Fortran
- Java
- JavaScript
- PHP
- Python
- Ruby
